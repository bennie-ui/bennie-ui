# Subgrid(col)

```
<Section grid={{ flow: 'col', templates:  { columns: '4'} }}>
    <Section>Item $</Section>*5
    <Section grid={{ templates: { column: 'subgrid' } }}>
      <Section 
        grid={{ start: { rows: '4' } }} 
        colors={{ background: { color: 'purple'} }}>
    </Section>
</Section>

```
