# Subgrid(row)

```
<Section grid={{ flow: 'col', templates:  { rows: '4'} }}>
    <Section>Item $</Section>*5
    <Section grid={{ templates: { rows: 'subgrid' } }}>
      <Section 
        grid={{ start: { rows: '2' } }} 
        colors={{ background: { color: 'purple'} }}>
    </Section>
</Section>
```
