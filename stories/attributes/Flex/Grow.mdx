# Flex 

## Grow(1):row

Example
```
<Section flex={{ direction: "row" }}>
    <Section>Item 1</Section>
    <Section flex={{ grow: '1' }}>Item 2</Section>
    <Section>Item 3</Section>
  </Section>

```
