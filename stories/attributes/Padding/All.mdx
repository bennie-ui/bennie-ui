# Demo of supported padding values
Tailwind supports values up to margin(96), initially phoenix-ui will go as far as 40.

## Example
```
<Section margin={{ all: '0' }}Content</Section>
```
## Types
[See Margin Values](https://github.com/bolanosdev/phoenix-ui/blob/main/libraries/types/spacing/margin.types.ts)
