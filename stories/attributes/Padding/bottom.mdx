# Demo of supported padding values
Tailwind supports values up to margin(96), initially bennie-ui will go as far as 40.

## Example
```
<Section padding={{ bottom: '0' }}Content</Section>
```
## Types
[See Margin Values](https://github.com/bolanosdev/bennie-ui/blob/main/libraries/types/spacing/padding.types.ts)
